\resumo{Resumo}{Escrever soluções para problemas computacionais é algo extremamente individual, cada pessoa possui a sua própria maneira de construir algoritmo em uma determinada linguagem de programação, este tipo de conduta pode causar diversos problemas quando se tem várias pessoas trabalhando no mesmo projeto. Este trabalho tem por finalidade retratar a importância de se utilizar um estilo de codificação para o processo de desenvolvimento de um software e apresentar um estilo de codificação que foi criado para a linguagem Python, que é denominado PEP8. Para a validação deste estudo é analisado um repositório de projetos \textit{open-source}, o qual é mensurado qual o nível de adoção do estilo mencionado. A medição do nível de adoção é dada através da execução de ferramentas descritas neste trabalho que analisa cada código-fonte que é coletado através de rotinas criadas especificamente para esta tarefa.
}

\vspace*{2cm}

\resumo{Abstract}{Write solutions to computational problems is extremely individual, each person has their own way to build algorithm in a particular programming language, this kind of conduct can cause various problems when you have multiple people working on the same project. This study aims to portray the importance of using a coding style to the process of developing a software and submit a coding style that was created for the Python language, which is called PEP8. To validate this study is analyzed a project repository \ textit {open source}, which is measured which the level of adoption of that style. Measuring the level of adoption is given by running tools described in this paper that analyzes each source code that is collected through routines designed specifically for this task.}
