\resumo{Resumo}{Escrever soluções para problemas computacionais é algo extremamente individual, cada pessoa possui a sua própria maneira de construir algoritmo em uma determinada linguagem de programação, este tipo de conduta pode trazer diversos problemas quando se tem várias pessoas trabalhando no mesmo projeto. Com isto este trabalho tem por finalidade retratar a importância de se utilizar um estilo de codificação para o processo de desenvolvimento de um programa de computador e apresenta um estilo de codificação que foi criado para a linguagem Python, que é denominado PEP8, para a validação deste estudo é analisado um repositório de projetos open-source, o qual é mensurado qual o nível de adoção do estilo mencionado. A medição do nível de adoção é dada através da execução de ferramentas descritas neste trabalho que analisam cada código fonte que é coletada através de rotinas criadas especificamente para esta tarefa.
}

\vspace*{2cm}

\resumo{Abstract}{Write solutions to computational problems is extremely individual, each person has the their own way of building algorithm in a particular programming language, such conduct can bring many problems when you have multiple people working on the same project. with that this study aims to portray the importance of using a coding style for process of developing a computer program and displays a coding style that was created for the Python language, which is called PEP8, for the validation of this study is analyzed a repository of open-source projects, which is measured which the level of adoption of that style. The measurement of the level of adoption is given by running tools described in this work analyze each source code that is collected through routines designed specifically for this task.}
